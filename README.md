# CODVEDA_PROJECT

A collection of Machine Learning projects and algorithm implementations completed during my internship at Codveda Technologies. This repository demonstrates the practical application of various supervised learning algorithms using Python and Scikit-learn.

## 📌 Overview

This repository contains multiple Machine Learning projects covering classification and regression tasks. Each notebook includes data preprocessing, model training, evaluation, and result analysis.

## 🚀 Projects Included

### 1. Linear Regression

* Predictive modeling using Linear Regression
* Data preprocessing and visualization
* Model evaluation using regression metrics

### 2. House Price Prediction

* Real-world regression problem
* Feature analysis and prediction of house prices

### 3. K-Nearest Neighbors (KNN)

* Classification using distance-based learning
* Performance evaluation and comparison

### 4. Decision Tree Classifier

* Tree-based classification model
* Feature importance analysis

### 5. Decision Tree Regressor

* Regression using Decision Trees
* Prediction and performance assessment

### 6. Random Forest

* Ensemble learning technique
* Improved prediction accuracy through multiple trees

### 7. Support Vector Machine (SVM)

* Classification using Support Vector Machines
* Hyperplane-based decision making

### 8. Naive Bayes

* Probabilistic classification algorithm
* Suitable for classification tasks and pattern recognition
